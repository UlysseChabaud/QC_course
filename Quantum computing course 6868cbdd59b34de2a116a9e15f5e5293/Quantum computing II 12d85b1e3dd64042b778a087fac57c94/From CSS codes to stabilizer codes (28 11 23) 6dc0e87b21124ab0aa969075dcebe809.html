From CSS codes to stabilizer codes (28/11/23)

In this lecture we generalize the method to construct quantum codes from classical ones that lead to Shor’s code. The resulting codes are known as Calderbank-Shor-Steane codes, or CSS. Their construction highlights an additional condition that the starting classical codes need to satisfy. This can be understood using the language of stabilizers, which we study next. Although introduced by Daniel Gottesman to study QEC and fault tolerance, the stabilizer language has proven to be a very powerful tool to tackle many other problems in quantum computing and quantum information theory. We will see an example of this by proving the Gottesman-Knill theorem for the efficient classical simulation of a subset of quantum computations.

We have mentioned that any stabilizer state can be obtained from ∣000…⟩|000\ldots\rangle∣000…⟩﻿ by applying a Clifford circuit. Furthermore, any Clifford circuit can be compiled in O(n2)\mathcal{O}(n^2)O(n2)﻿ CXCXCX﻿, Z\sqrt{Z}Z​﻿ and Hadamard gates. At any point in the circuit, the state remains a stabilizer state, which is specified by exactly nnn﻿ Pauli operators (the stabilizer generators). These operators are updated after application of each gate in the circuit. In the worst case, we need to update each single-qubit operator appearing in each stabilizer generator, which results in n2n^2n2﻿ updates. We have seen that as long as we perform Pauli measurement on stabilizer states, the outcomes are either deterministic (if the observables commute with all stabilizer generators at the time of measurement) or completely at random. In the latter case, measurement outcomes can be simulated by drawing random bits, and the post-measurement state is obtained through the update rules above. Overall, such an algorithm results in O(n4)\mathcal{O}(n^4)O(n4)﻿ operations.

This theorem was quite surprising when it was first discovered, as it challenges the intuition that advantage in quantum computations must come from entanglement. Indeed, stabilizer states include highly entangled states, such as code states of quantum error correcting codes, and yet anything that preserves their “stabilizer status” cannot provide any advantage.

The stabilizer formalism is exploited for the efficient simulation of many quantum information protocols (such as teleportation, communication protocols, stabilizer computations, …), such as in Qiskit’s extended stabilizer simulator.

So where does the advantage come from? There is no single way (so far) to separate exactly which resources allow for advantage (although contextuality is a good candidate). In a model where we have a subset of our universal gate set composed by Clifford gates, where we only start from computational basis states and perform computational basis measurements, we must look at where the theorem fails: namely non-Clifford gates.
